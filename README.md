# mobile-webdriverio-browserStack

>This repo consist of android, iOS and Web automation.

### App Used

- **Android**: APIDemos
- **iOS**: UIKitCatalog, MVCTodo
- **Web**: webdriverio.com

### Cloud: 
- Browserstack
### Framework: 
- Mocha

### To run test on local

#### Android:
``` npm run android ```

#### iOS:
``` npm run ios ```

#### web:
``` npm run web ```
